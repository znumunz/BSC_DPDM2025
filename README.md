## Intro (คาบที่ 1)

### เกณฑ์การให้คะแนนและรูปแบบการเรียนการสอน
- อาจารย์แจ้ง **เกณฑ์การให้คะแนน งาน/โปรเจคที่ต้องทำ** ของวิชา
- การเรียนการสอนในวิชานี้จะเน้นทฤษฎี **Data Mining** มากกว่า **Basic Programming** ของเทอมก่อน
- การส่งงาน **Coding** ต้องใช้ **GitHub**
- สามารถอ่านเนื้อหาใน **Textbook: Data Mining: Concepts and Techniques** ได้โดยไม่ต้องเข้าเรียน แต่จะไม่ได้คะแนน **เข้าเรียน** และความรู้เรื่อง **Coding**

### เนื้อหาที่เรียนในคาบแรก
- แนะนำหนังสือที่ดีมากๆ สามารถอ่านได้โดยไม่ต้องเข้าเรียน
- **Data Mining** เกิดขึ้นจากความต้องการหาประโยชน์จากข้อมูลมหาศาล  
  - เปรียบเหมือน **เหมืองแร่/เหมืองทอง**
- กำเนิดของ Data Mining เกิดประมาณ **40-50 ปีก่อน**  
  - ตัวอย่าง: ร้านสะดวกซื้อพบข้อมูลลูกค้า **ซื้อผ้าอ้อม + เบียร์** คู่กัน → รู้ว่าลูกค้าเป็น **พ่อบ้าน**  
  - นำไปสู่การทำ **Insight Analysis** เพื่อปรับกลยุทธ์ให้ตรงกลุ่มลูกค้า แทนการทำโปรโมชั่นแบบกลางๆ
